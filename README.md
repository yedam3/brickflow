## 👉🏻 개요

**BrickFlow**는 레고 제조 현장의 **생산 관리, 작업 지시, 품질 관리, 설비 상태 모니터링** 등  
다양한 업무를 누구나 직관적이고 쉽게 사용할 수 있도록 설계된 **MES (Manufacturing Execution System)** 입니다.

---

## 👥 팀원 소개

| 이름 | 역할 |
|------|------|
| 🧱 신현욱 (팀장) | 자재 관리 |
| ⚙️ 복성민 (부팀장) | 생산 담당 |
| 💵 오세경 | 영업 담당 |
| 🔧 정은총 | 설비 담당 |
| 🔍 김지수 | 검수 및 관리자 |
