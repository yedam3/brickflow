#  🔥 PROJECT: BrickFlow 🔥

## 👉🏻 개요

본 프로젝트는 복잡한 제조 현장의 생산 관리, 작업 지시, 품질 관리, 설비 상태 모니터링 등 다양한 업무를 누구나 직관적이고 쉽게 사용할 수 있도록 설계된 구독형 MES(Manufacturing Execution System) 프로그램을 개발하는 것을 목표로 합니다.

## 🙋🏻‍♂️ 팀원

* 신현욱(팀장) - 자재 🧱
* 복성민(부팀장) - 생산 ⚙️
* 오세경 - 영업 💵
* 정은총 - 설비 🔧
* 김지수 - 검수 🔍
